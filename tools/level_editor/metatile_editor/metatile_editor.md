# Metatile Editor

This tool is part of the **Great RA level editor** and is used to construct and manage metatile data for the game.

---

## Overview

The editor operates in multiple modes, allowing you to:

* Select individual tiles (1×1)
* Construct 2×2 metatiles
* Build larger 8×2 structures
* Assign attributes and tile types

Press **Select** at any time to cycle between modes.

---

## Modes

### Mode 0 — 1×1 Tile Selection

* Use the **D-Pad** to scroll and select a tile
* The tile under the cursor becomes the active 1×1 tile when switching modes

---

### Mode 1 — 2×2 Tile Editor

* Use the **D-Pad** to select one of the four tiles in the 2×2 block
* Press **A** to place the current 1×1 tile
* Press **B** to cycle the tile type (values `0–7`)

---

### Mode 2 — 8×2 Tile Editor

* Use the **D-Pad** to select a 2×2 block within the 8×2 structure
* Press **A** to place the current 2×2 metatile
* Press **B** to change the attribute of the selected 2×2 tile

---

### Mode 3 — 2×2 Tile Selection

* **Up / Down** – change the current 2×2 metatile
* **Left / Right** – toggle increment mode:

  * `$10` steps
  * `$01` steps

---

### Mode 4 — 8×2 Tile Selection

* Same behavior as Mode 3
* Applies to selecting 8×2 metatiles

---

## Palette Editing

* Palette values can be modified using the hex editor in an emulator (e.g., FCEUXD)
* Palette data is located at:

  ```text
  $30–$3F
  ```
* The editor supports **4 palettes**
* The active mode is highlighted (assuming palettes are visually distinct)

### Saving Palette

* Press **Start** to save palette data
* Palette will persist across sessions

---

## Data Storage

All metatile and palette data is stored in the `.SAV` file generated by the ROM.

### 2×2 Tile Data

```text
$7000–$70FF  Top Left
$7100–$71FF  Top Right
$7200–$72FF  Bottom Left
$7300–$73FF  Bottom Right
```

### 8×2 Tile Data

```text
$7400–$74FF  Far Left
$7500–$75FF  Mid Left
$7600–$76FF  Mid Right
$7700–$77FF  Far Right
```

### Attributes and Types

```text
$7800–$78FF  8×2 Attributes
$7900–$79FF  2×2 Tile Types
```

### Palette

```text
$6D00–$6D10
```

---

## Workflow Notes

* The editor writes directly to the `.SAV` file
* It is recommended to use a **shared level save file** for metatile editing
* This allows reuse across ROM builds without recompilation
